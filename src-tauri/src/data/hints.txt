how to access the db and its data: 
open terminal in folder where products db is
type: sqlite3 products.db

you now have access to do whatever here. just make sure you add a semi colon in the end

for the orders data it is saved in the root /data folder which you can then access by:
open terminal in folder where products db is
type: sqlite3 ordering_system_data.db

you now have access to all of its data. make sure you use "orders" for the query instead of the file name
